Capabilities

Capabilities describe optional features that a AbstractChatClient may support. The trait-like system uses singleton types (e.g., HasStreaming) and query functions (e.g., supports_streaming) so agent code can adapt to provider features at runtime.

Core Types

AgentFramework.Capability — Type

Base type for all capability traits.

AgentFramework.NoCapability — Type

Sentinel type indicating a capability is not supported.

Capability Traits

AgentFramework.HasEmbeddings — Type

Indicates a client supports text embedding generation.

AgentFramework.HasImageGeneration — Type

Indicates a client supports image generation.

AgentFramework.HasCodeInterpreter — Type

Indicates a client supports code interpretation/execution.

AgentFramework.HasFileSearch — Type

Indicates a client supports file search.

AgentFramework.HasWebSearch — Type

Indicates a client supports web search.

AgentFramework.HasStreaming — Type

Indicates a client supports streaming responses.

AgentFramework.HasStructuredOutput — Type

Indicates a client supports structured output (JSON schema responses).

AgentFramework.HasToolCalling — Type

Indicates a client supports tool/function calling.

Query Functions

AgentFramework.has_capability — Function
has_capability(client::AbstractChatClient, cap_fn::Function) -> Bool

Check whether client has the capability queried by cap_fn.

AgentFramework.supports_embeddings — Function

Return true if client supports text embeddings (i.e. embedding_capability(typeof(client)) is not NoCapability).

AgentFramework.supports_image_generation — Function

Return true if client supports image generation.

AgentFramework.supports_code_interpreter — Function

Return true if client supports code interpretation/execution.

AgentFramework.supports_file_search — Function

Return true if client supports file search.

AgentFramework.supports_web_search — Function

Return true if client supports web search.

AgentFramework.supports_streaming — Function

Return true if client supports streaming responses.

AgentFramework.supports_structured_output — Function

Return true if client supports structured output (JSON-schema-constrained responses).

AgentFramework.supports_tool_calling — Function

Return true if client supports tool/function calling.

Capability Introspection

AgentFramework.list_capabilities — Function
list_capabilities(client::AbstractChatClient) -> Vector{Symbol}

Return a list of capability symbols supported by client.

AgentFramework.require_capability — Function
require_capability(client::AbstractChatClient, cap_fn::Function, operation::String)

Throw a ChatClientError if client does not support the capability queried by cap_fn.

Capability Actions

AgentFramework.get_embeddings — Function
get_embeddings(client::AbstractChatClient, texts::Vector{String}; model=nothing) -> Vector{Vector{Float64}}

Generate embeddings for the given texts. Only available for clients with HasEmbeddings.

AgentFramework.generate_image — Function
generate_image(client::AbstractChatClient, prompt::String; size=nothing, quality=nothing) -> Any

Generate an image from a text prompt. Only available for clients with HasImageGeneration.

Capability Constructors

AgentFramework.embedding_capability — Function

Trait dispatcher returning the embedding capability of a chat client type. Returns NoCapability() by default; provider implementations should override to return a concrete Capability singleton when supported.

AgentFramework.image_generation_capability — Function

Trait dispatcher returning the image-generation capability of a chat client type. Defaults to NoCapability().

AgentFramework.code_interpreter_capability — Function

Trait dispatcher returning the code-interpreter capability of a chat client type. Defaults to NoCapability().

AgentFramework.file_search_capability — Function

Trait dispatcher returning the file-search capability of a chat client type. Defaults to NoCapability().

AgentFramework.web_search_capability — Function

Trait dispatcher returning the web-search capability of a chat client type. Defaults to NoCapability().

AgentFramework.streaming_capability — Function

Trait dispatcher returning the streaming capability of a chat client type. Defaults to NoCapability().

AgentFramework.structured_output_capability — Function

Trait dispatcher returning the structured-output capability of a chat client type. Defaults to NoCapability().

AgentFramework.tool_calling_capability — Function

Trait dispatcher returning the tool-calling capability of a chat client type. Defaults to NoCapability().
