Exceptions

A structured exception hierarchy for fine-grained error handling. All exceptions derive from AgentFrameworkError. Catch specific subtypes to handle agent errors, chat client errors, tool errors, middleware errors, or workflow errors independently.

Base Error

AgentFramework.AgentFrameworkError — Type
AgentFrameworkError <: Exception

Base exception for the Agent Framework. All framework-specific errors derive from this.

Fields

  • message::String: Human-readable error description.
  • inner::Union{Nothing, Exception}: Optional wrapped cause.

Agent Errors

AgentFramework.AgentError — Type

Base class for all agent exceptions.

AgentFramework.AgentInvalidAuthError — Type

Raised by an agent when authentication with the underlying provider fails.

AgentFramework.AgentInvalidRequestError — Type

Raised by an agent when the request to the underlying provider is invalid.

AgentFramework.AgentInvalidResponseError — Type

Raised by an agent when the response from the underlying provider cannot be parsed or is malformed.

Chat Client Errors

AgentFramework.ChatClientError — Type

Base class for all chat client exceptions.

AgentFramework.ChatClientInvalidAuthError — Type

Raised by a chat client when authentication with the LLM provider fails.

AgentFramework.ChatClientInvalidRequestError — Type

Raised by a chat client when the request to the LLM provider is invalid (bad model name, oversized input, etc.).

Tool Errors

AgentFramework.ToolError — Type

Base class for all tool-related exceptions.

AgentFramework.ToolExecutionError — Type

Raised when a tool's execution function throws an unhandled exception.

Content Errors

AgentFramework.ContentError — Type

Raised when constructing or manipulating Content items in an inconsistent way.

Middleware Errors

AgentFramework.MiddlewareError — Type

Raised when a middleware function in the pipeline throws or returns an invalid value.

Workflow Errors

AgentFramework.WorkflowError — Type

Raised when a workflow encounters a structural error (cycles, dead ends, type mismatches, etc.).

Declarative Errors

AgentFramework.DeclarativeError — Type

Raised when a declarative agent specification fails to parse or validate.

User Input Errors

AgentFramework.UserInputRequiredError — Type
UserInputRequiredError

Raised when a tool wrapping a sub-agent requires user input to proceed. Carries the request contents so the parent can propagate them.
